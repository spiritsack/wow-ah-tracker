# WoW TBC Anniversary — AH Price Tracker

A single-page app for logging and analysing Auction House prices in World of Warcraft TBC Anniversary.

No backend. All data stored in your browser's `localStorage`.

## Features

- Log min buyout and market price per item
- Price history chart per item
- Weekday trend analysis (averages, ranges)
- All-items spread chart (market price vs min buyout) to spot underpriced items
- JSON export/import for backup

## Dev

```bash
npm install
npm run dev
```

## Stack

Vite · React 18 · TypeScript · Chart.js
